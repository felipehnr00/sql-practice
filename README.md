# SQL Practice

Repositório dedicado aos meus estudos e exercícios de SQL voltados para análise de dados e bancos de dados relacionais.

## Conteúdos estudados

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* INNER JOIN
* Funções de agregação
* Manipulação de dados

## Objetivo

Praticar consultas SQL e desenvolver habilidades voltadas para análise, organização e tratamento de dados.
